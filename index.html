Setup

  1. Set STRAVA_CLIENT_ID and STRAVA_CLIENT_SECRET on your backend server.
  2. Set Strava Authorization Callback Domain to your frontend domain (for example localhost or yourname.github.io).
  3. Set STRAVA_REDIRECT_URI to your frontend URL exactly (for example http://localhost:8080/ or https://yourname.github.io/repo/).

If this page is on GitHub Pages, set window.RUNNING_TRACKER_API_BASE in config.js to your deployed backend URL.
