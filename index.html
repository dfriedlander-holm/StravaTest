Setup

  1. Set STRAVA_CLIENT_ID and STRAVA_CLIENT_SECRET before starting the server.
  2. In your Strava app settings, set Authorization Callback Domain to localhost.
  3. Set STRAVA_REDIRECT_URI to exactly http://localhost:8080/ (or your chosen host/port).
